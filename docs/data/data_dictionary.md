## Base de datos 1

La base de datos utilizada cuenta con 87.000 imágenes, con 29 clases específicas, donde 26 de ellas se refieren a letras de la A-Z, y 3 adicionales se refieren a las clases ESPACIO, ELIMINAR y NADA.

Esta base de datos fue obtenida de Kaggle (https://www.kaggle.com/datasets/grassknoted/asl-alphabet)
